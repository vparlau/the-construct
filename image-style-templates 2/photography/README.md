# Photography Styles

This category contains image generation prompt templates for photographic and camera-based styles.

## What belongs here

- Realistic photographic styles that mimic camera-captured images
- Stylized photography genres (noir, neon noir, chiaroscuro)
- Any style that emphasizes photographic techniques, lighting, and composition

## Styles included

- Realistic Photographic
- Noir / Chiaroscuro
- Neon Noir
