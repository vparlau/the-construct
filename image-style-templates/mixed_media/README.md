# Mixed Media Styles

This category contains image generation prompt templates for styles that combine multiple techniques, mediums, or don't fit neatly into a single category.

## What belongs here

- Styles that blend multiple artistic techniques
- Concept art and matte painting
- Collage and mixed media techniques
- Styles that combine digital and traditional elements
- Specialized craft-based styles (mosaic, papercut, claymation)

## Styles included

- Concept Art / Matte Painting
- Collage / Mixed Media
- Cyberpunk / Neon Futurism
- Mosaic / Stained Glass
- Papercut / Paper Craft
- Claymation / Stop Motion
